# ProjetoFinalHTML
Projeto Final do Curso de Des. HTML - Prof Raphael Jesus

Site Chalé Hotel

Usadas as tecnologias HTML5, CSS e JavaScript.

O site utiliza recursos como formulários, acessibilidade, imagens e layout líquidos.

O site apresenta o hotel e suas características, permite contato através de formulários e reservas a partir do site selecionando datas de check-in e check-out.
